# Masterfile Automation - Target

Ultra-fast Excel masterfile automation tool for Target marketplace. This tool fills data from onboarding sheets into the Target masterfile template while preserving all formatting, formulas, and other sheets.

## Features

- ✅ Fast XML-based writing (seconds, not minutes)
- ✅ Preserves all sheets (Bulk Product Data, Dietary Supplements, Report Details)
- ✅ Maintains formatting, formulas, and macros
- ✅ Smart column mapping with fuzzy matching
- ✅ Supports both .xlsx and .xlsm formats
